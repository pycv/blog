## 欢迎来到我的博客

这是使用 Hugo 静态网站生成器搭建的第一篇博客文章。

### 关于 Hugo

Hugo 是一个用 Go 语言编写的快速、现代的静态网站生成器，它具有以下特点：

- **快速构建**：Hugo 可以在几秒钟内构建整个网站
- **易于使用**：简单的命令行工具和配置
- **主题丰富**：拥有大量精美的主题可供选择
- **支持 Markdown**：使用 Markdown 编写内容，简单高效

### 本站使用的主题

本博客使用 PaperMod 主题，这是一个简洁、美观且功能强大的 Hugo 主题。

```bash
# 安装主题
git submodule add --depth=1 https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod
```

### 开始写作

现在可以开始创建更多文章了：

```bash
# 创建新文章
hugo new content/posts/my-new-post.md
```
