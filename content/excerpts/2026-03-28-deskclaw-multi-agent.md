## 背景

DeskClaw 是一个开源的多 Agent 协作平台，让企业可以像管理真实团队一样管理 AI 员工。通过可视化画布配置多个 AI Agent 协同工作，实现业务流程自动化。项目完全开源，支持 Docker 一键部署。

## 核心观点

- **AI 公司可视化**：通过 3D/2D 画布管理 AI 员工，中央黑板实时查看任务状态、讨论和文件
- **动态技能市场**：技能被打包成"基因"，通过使用次数、人类反馈和 Agent 互评三个维度动态评分，高质量基因自动浮现
- **企业级能力**：内置权限控制、操作日志、审计追溯、效能度量，让 AI 在企业中真正可管理、可审计

## 补充要点

- **多模型支持**：MiniMax、Kimi、OpenRouter、阿里云、火山引擎等，切换模型时记忆和上下文不丢失
- **即时通讯集成**：一键接入飞书、钉钉、企微、Slack，AI 员工直接出现在日常工作流
- **电商团队案例**：营销规划 + 爆款抓取 + AIGC 生成 + 内容投放 + 内容审查，全自动化营销链路
- **Docker 一键部署**：三条命令即可启动，访问 localhost 即可使用

## 为什么值得关注

过去两年 AI Agent 行业从比模型能力、比任务编排，演进到比谁能帮助企业稳定运转多 Agent 系统。DeskClaw 团队在电商营销场景有长期实践经验，这套系统从真实业务中生长出来，自己就是第一个用户。如果你正在考虑多 Agent 协作落地，这个开源项目值得体验。
