## 背景

中科大 20 岁学生 BaiFu 用 10 天 VibeCoding 出两个开源项目：BettaFish（多 Agent 舆情分析）和 MiroFish（AI 预测引擎），双双登顶 GitHub Trending。后者获盛大陈天桥 3000 万投资。文章分享了 VibeCoding 的方法论和对「超级个体」的思考。

## 核心观点

- **MiroFish 做什么**：通过提取现实世界的种子信息，构建平行数字世界，让成千上万个具备独立人格、长期记忆的智能体进行社会演化，推演事件走向（如红楼梦续写、企业战略分析）
- **VibeCoding 落地流程**：Figma 画草图 → Google AI Studio 做前端 demo → 拆分模块交给 AI IDE 开发；前端推荐 Gemini 3 Pro（有灵气），后端工程用 Claude（稳定性）
- **多 Agent 并行策略**：同时开 8 个 agent 干一样的活，效率飞快，还能直观摸到每个模型的能力边界

## 补充要点

- BettaFish 一周暴涨 20k star，MiroFish 目前 5.7k star
- 红楼梦案例：从 15 万字原著生成 905 个实体节点、3822 条关系边、580 个人设，成本约 14 元
- 刹车系统很重要：git 管理代码 + 写好文档，防止改了 A 影响 B
- 深度 code review：一行行审计 AI 代码，理解它的思考过程，纠正关键误区
- 开源选题建议：从垂直领域切入积累口碑，再面向大众；做减法，不追求大而全
- 营销素材提前准备：演示视频等能让「别人帮你宣传」的内容
- 代码是冷的，故事是热的——学会讲故事是独立开发者的必修课
